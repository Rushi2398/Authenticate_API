# Authenticate_API

## Setup

Install typescript prisma and run the following commands to get the dependencies:

```bash
npm install --save-dev typescript @types/node ts-node prisma
```

Generate a TypeScript configuration file (tsconfig.json) using the TypeScript compiler. Run the following command:

```bash
npx tsc --init
```

Initialize Prisma in your project by running the following command:

```bash
npx prisma init
```

Generate the Prisma Client by running:

```bash
mpx prisma generate
```
